# Contributing to Neurix Token

## Disclaimer
By contributing, you agree that this is not an investment solicitation and that contributions are under the project's liability terms. The project maintainers are not responsible for financial advice or investment outcomes.

## Workflow
1. Fork the repo and create a feature branch (`feature/your-feature`).
2. Commit changes with clear messages.
3. Push to your fork and open a PR against `main`.
4. At least one reviewer approval is required for merge.

## Investor Information
We welcome feedback from the community. Investment inquiries should be directed to **palermoinsta1@gmail.com**. 
- Please include your organization details.
- Outline your interest in token distribution or partnerships.
- We will respond within 3 business days.

## Code Style
- Solidity: follow Solhint rules.
- JS/TS: follow ESLint + Prettier.

## Testing
Every new feature must include unit tests.
